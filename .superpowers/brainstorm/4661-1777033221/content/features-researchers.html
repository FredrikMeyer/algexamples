Potential features for researchers

Computation & verification

A

CAS code snippets

Attach Macaulay2 / SageMath code blocks to examples. A researcher can copy-paste to verify or extend the computation. E.g. "compute the Hilbert series of the twisted cubic in Macaulay2".

B

Commutative diagrams

Render commutative diagrams inline using TikZJax or quiver-style markup. Essential for exact sequences, morphisms, and functors — currently impossible with KaTeX alone.

C

Hodge diamond visualisation

For varieties with known Hodge numbers, render the Hodge diamond as an SVG alongside the properties table. Makes h^{pq} data scannable at a glance.

Navigation & cross-referencing

D

Automatic backlinks

"Referenced by" section computed at build time — if example B lists A in its related field, A's page automatically shows B as a backlink. No manual maintenance.

E

Stacks Project tag links

Special syntax like [SP 01AB] renders as a linked tag with hover tooltip showing the statement. Direct bridge to the authoritative reference.

F

"Is an example of" concept tags

Tag examples with the abstract concepts they instantiate — e.g. twisted cubic: rational curve · non-complete-intersection · Veronese embedding. Separate from free-form tags; concepts are a controlled vocabulary linking to a definition.

G

Side-by-side variety comparison

Select 2–3 varieties and view their properties tables side-by-side. Useful for understanding what distinguishes similar objects (e.g. twisted cubic vs rational normal curve in ℙ⁴).

Authoring & presentation

H

Site-wide LaTeX macros

Define macros once (\PP → \mathbb{P}, \OO → \mathcal{O}, \Spec → \operatorname{Spec}) and use them in any entry. Consistent notation across the whole database.

I

Export to LaTeX / PDF

Each example page has a "Download as .tex" button. The source markdown is already close to LaTeX — this makes it easy to include examples in notes or papers.

J

Notation / conventions page

A static page declaring site-wide conventions: field is always algebraically closed, schemes are Noetherian, etc. Resolves ambiguity for readers who land on a specific example directly.
