Design: Site Structure

Pages generated at build time from Markdown files, served as static HTML

Route tree
/ — Homepage: search bar + recent additions + browse by field
/examples — Full listing, filterable by type/field/tags
/examples/$slug — Individual example (the permalink)
/tags/$tag — All examples with a given tag
/fields/$field — All examples in a field (e.g. commutative algebra)
/types/$type — Browse by type: variety · computation · counterexample
